// About Me

Building quality into every release.

I'm a QA professional passionate about delivering bug-free, performant software. Over the past 7+ years, I've worked across domains including government e-commerce (GeM), automotive (Maruti Suzuki), and OTT streaming platforms (Roku, Apple TV, Xbox).

My approach combines hands-on manual testing with robust Selenium automation, API validation using Postman, and debugging through Charles Proxy. I thrive in Agile environments where continuous integration and rapid release cycles demand both precision and speed.

Beyond testing, I build tools that make work faster. My AI Resume Workflow project uses Claude AI to automatically optimize resumes for ATS systems, generating professional .docx and LaTeX outputs.
